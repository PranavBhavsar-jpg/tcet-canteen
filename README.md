TCET Canteen is a frontend-based food ordering web application designed to
digitize the traditional college canteen system. This project allows users
to browse food items, view prices, and add items to a cart through an
interactive and responsive user interface.

The project focuses on frontend development while simulating backend
operations using JavaScript. All cart management and order handling are
performed using JavaScript arrays and functions, making it ideal for
academic and beginner-level full stack understanding.

## Features
- Modern and responsive UI
- Food item listing with prices and preparation time
- Add to cart functionality
- Backend logic simulation using JavaScript
- Beginner-friendly and well-structured code

## Tech Stack
- HTML
- CSS
- JavaScript

## Project Type
Academic Mini Project / Frontend Simulation Project

## Future Enhancements
- User authentication
- Real backend integration (Node.js & MongoDB)
- Online payment system
- Order tracking system

## Note
This project is developed for learning and academic purposes. Backend
functionality is simulated without using a real database or server.
